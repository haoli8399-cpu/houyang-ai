## 一、架构冻结

| # | 规则 | 说明 |
|---|------|------|
| 1 | FreeSWITCH 是唯一的电话接入层 | 不引入其他 SIP Server |
| 2 | ESL 端口固定 8022（127.0.0.1） | 不改成 8021 或其他 |
| 3 | AI 引擎入口固定 `ai_pipeline.py` | 不拆分微服务架构 |
| 4 | ASR = faster-whisper 本地部署 | 不切换云端 ASR |
| 5 | TTS = CosyVoice 本地部署（HTTP 9880） | 不切换云端 TTS |
| 6 | LLM = DeepSeek API | 不切换其他 LLM 供应商 |
